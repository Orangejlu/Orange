# Orange
using Oracle

##“选课系统”
系统分为四大模块：管理员，教务，教师，学生。    
管理员可以发布公告，添加院系，添加教务用户；   
教务用户可以添加教师，添加学生，管理课程；   
教师用户可以查看课表，查看学生名单，上成绩；   
学生用户可以选课，查课表，查成绩。
